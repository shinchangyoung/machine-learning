4월 28일 
탐색적 데이터 분석(EDA ( Exploratory Data Analysis))의 이해
데이터를 탐구하고 숨겨진 안사이트를 발견하자
인사이트: 데이터를 통해 얻은 통찰이나 깨달음

✅EDA의 개념을 이해할 수 있다
EDA의 주요 과정과 도구를 활용할 수 있다

EDA란 무엇인가

✅EDA정의 
-데이터를 분석하고 이해하는 초기과정
 데이터가 어떤 구조를 가지고 있는지, 어떤 패턴이 숨어 있는지등

-데이터의 구조, 패턴,이상치 탐구
  데이터 안에 이상치나 결측치 문제가 있는지 확인한다

✅EDA의 중요성
- 데이터 품질 확인하기
	데이터가 깨끗한지, 잘못된 값이 없는지 확인

-데이터 기반의 의사결정을 지원한다
	데이터를 잘 이해하면 더 나은 비즈니스나 연구 결정 가능

-보이지 않더 중요한 패턴이나 관계를 찾을 수 있다

✅ EDA의 3가지 주요 과정

1.데이터 이해 : 데이터 유형 확인 기술통게 분석 결측치/ 이상치 탐구
		     이상치: 비정상적인 값, 한 값만 너무크거나 작은 그런 통계값
			데이터이해단계가 왜 필요할까?
				>> 데이터에있는 결측값과 이상치를 해결하면 분석결과가 정확하게 나올 수가 있다

2. 시각적 분석 : 더 잘 이해하기 위해 그래프와 차트를 활용한다
		ex) 산점도,히스토그램,박스플롯(이상치를 찾는데 탁월), 히트맵(상관관계를 시각적으로 확인가능)


3.데이터 변환 : 분석하기 쉬운 형태로 바꾸는 단계
		ex) 스케일링 : 데이터를 일정한 범위로 조정해주는 역할 ->키와 몸무게를 비교하기 위해서 같은 범위로 변환하거나
			정규화 (Normalization)	데이터의 최소값을 0, 최대값을 1로 바꾸는 것
			표준화 (Standardization)	데이터의 평균을 0, 표준편차를 1로 바꾸는 것
			ex)표준화예시
			데이터 세트 1: [50, 51, 49, 50, 50]
			데이터 세트 2: [30, 70, 10, 90, 50]
			둘 다 평균은 50.
			그런데 세트 1은 모두 평균 50 주변에 몰려있지? → 표준편차 작음
			세트 2는 엄청 널뛰고 있네? → 표준편차 큼	
		> 그래서 표준편차가 크면 "데이터가 퍼져 있다"고 하고, 작으면 "데이터가 모여 있다"고 한다.		

		   로그변화: 데이터가 너무크거나 작을 경우 데이터의 크기를 조절해준다
		    파생변수 생성 : 기존정보를 종합해서 -> 새로운 정보를 생성해준다


‼️강사의 EDA  설명
✅이 과정이 왜 필요할까 
	데이터의 결측치 이상치를 발견하고 변수관계를 파악 => 분석결과와 신뢰도를 높이기 위해서 EDA는 필수 적이다

✅이 과정을 어떻게 적용하나
	데이터의 기본정보 확인 -> 결측치/이상치 탐구 -> 데이터의 분포나 변수간의 관계를 시각화해준다 
	-> 데이터 변환을 준비한다

✅이 단계의 결과는 무엇인가
	품질 구조 명확히 이해가능
	중요한 인사이트 발견
	분석을 높임과 모델링의 성공 가능성을 높일수 있다

>> 수정본
✅ EDA가 필요한 이유
데이터의 결측치와 이상치를 찾아내고,

변수들 간의 관계를 파악하여,

분석 결과의 신뢰도를 높이기 위해 EDA는 필수적인 과정이다.

✅ EDA는 어떻게 진행하나
데이터 기본 정보 확인
(데이터 크기, 컬럼 종류, 데이터 타입 등)

결측치와 이상치 탐색
(비어있는 데이터나 비정상적인 값을 찾기)

데이터 분포와 변수 간 관계 시각화
(히스토그램, 박스플롯, 산점도 등으로 흐름과 관계를 이해)

데이터 변환 준비
(정규화, 표준화, 인코딩 등 필요한 전처리 방향 결정)

✅ EDA 결과로 얻는 것
데이터의 품질과 구조를 명확히 이해할 수 있다.

데이터 속에서 중요한 인사이트를 발견할 수 있다.

분석의 정확도를 높이고, 모델링의 성공 가능성을 크게 높일 수 있다.


핵심: EDA는 데이터를 깊이 이해하고 문제를 미리 발견하여, 분석 신뢰도를 높이고 성공적인 모델링을 준비하는 필수 과정이다.

복습 
EDA(탐색적 데이터 분석) :  데이터를 분석하고 이해하는 초기과정 의미
데이터 분포 확인 : 히스토그램과 KDE로 데이터 분포 파악
이상치 탐지: box plot로 이상치 확인

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

data = { 'category' : np.random.choice(['A','B', 'C'], size=100),
          'value1' : np.random.randint(1, 100, size=100),
          'value2' : np.random.normal(loc=50, scale=15, size=100),
          'Date' : pd.date_range(start='2025-01-01', periods=100)}

df = pd.DataFrame(data)
df

#데이터 분포 획인
#히스토그램
plt.figure(figsize=(10, 6))
plt.hist(df['value1'], bins = 15, color='blue', edgecolor='black', alpha=0.7)
plt.xlabel('value1')
plt.ylabel('Frequency')
plt.grid()
plt.show()
